# ربات Forex

بازار واقعی این ربات Crypto Futures است. تحلیل از OKX گرفته می‌شود و اجرا روی Toobit انجام می‌شود.

## قانون‌های اصلی

- جهت اصلی فقط از 1D ارز گرفته می‌شود.
- BTC و ETH هم باید روی 1D هم‌جهت با ارز باشند.
- ورود دقیق روی 15M و 5M است.
- TP فقط یکی است.
- SL فقط یکی است.
- حداقل فاصله Entry تا TP روزانه باید 3٪ باشد.
- ریسک به ریوارد باید 1:2 باشد.
- toobit_client.py از فایل ارسالی کپی شده و دست‌کاری نشده است.
- برای خروج Toobit، trade_manager بعد از close چند ثانیه صبر می‌کند و اگر TOOBIT_PATH_CLOSE_CONFIRM تنظیم شده باشد درخواست Confirm Close را هم می‌فرستد.

## دستورات فارسی تلگرام

- /panel پنل ترید
- /trade_on روشن کردن ترید
- /trade_off خاموش کردن ترید
- /amount 10 تنظیم مبلغ معامله
- /leverage 10 تنظیم لوریج
- /max_positions 3 تنظیم حداکثر پوزیشن
- /stats نمایش آمار
- /reset_stats ریست آمار
- /delete_stats حذف آمار
