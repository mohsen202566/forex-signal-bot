# Forex Bot - نسخه تمیز اصلاح سیگنال اتومات

این نسخه برای اصلاح ریشه‌ای مشکل اسکن و سیگنال اتومات ساخته شده است.

## تغییرات اصلی

- `main.py` از نو تمیز شد.
- حلقه اسکن و مانیتور داخل loop فعال تلگرام با `asyncio.create_task` اجرا می‌شوند.
- استفاده مشکل‌دار از `Application.create_task` حذف شد.
- برای هر اسکن، لاگ کامل چاپ می‌شود:
  - شروع اسکن
  - تعداد سیگنال‌ها
  - تعداد رد شده‌ها
  - تعداد خطاها
  - ۱۰ دلیل اصلی رد شدن سیگنال‌ها
- همه سیگنال‌ها، حتی در حالت ترید خاموش یا اسلات پر، برای نتیجه‌گیری نمایشی ذخیره و مانیتور می‌شوند.
- پنل ترید اسلات واقعی، اسلات خالی و سیگنال‌های نمایشی باز را نشان می‌دهد.
- `toobit_client.py` دست‌نخورده و مطابق فایل آپلودی نگه داشته شده است.

## اجرای تمیز روی سرور

```bash
cd /root/forex-signal-bot
pkill -9 -f "main.py" || true

# فایل زیپ را داخل همین پوشه آپلود کن یا از مسیر دانلودت کپی کن
unzip -o forex_bot_auto_signal_clean_fix.zip
cp -r forex_bot_final/* .
cp -r forex_bot_final/.[!.]* . 2>/dev/null || true

source venv/bin/activate
pip install -r requirements.txt
python -m py_compile *.py
nohup venv/bin/python main.py > forex.log 2>&1 &
tail -f forex.log
```

