# اصلاح متنی و حذف فیلتر BTC/ETH

این نسخه:
- همه دکمه‌ها/InlineKeyboard را از ربات حذف می‌کند.
- فقط دستورهای متنی فارسی را می‌فهمد.
- شرط هم‌جهت بودن BTC و ETH را از قوانین سیگنال حذف می‌کند.
- main.py دیگر برای هر اسکن BTC/ETH روزانه نمی‌گیرد، پس اسکن سبک‌تر می‌شود.
- toobit_client.py تغییر نکرده است.

دستورهای اصلی:
- ترید
- ترید فعال
- ترید خاموش
- آمار
- تنظیم مبلغ 10
- تنظیم لوریج 10
- تنظیم حداکثر پوزیشن 3
- ریست آمار
- حذف آمار
