# Forex Root Fix

این نسخه دو مشکل اصلی را یک‌جا اصلاح می‌کند:

1. دستورات فارسی تلگرام:
   - ترید
   - ترید فعال
   - ترید خاموش
   - آمار
   - تنظیم مبلغ 10
   - تنظیم لوریج 10
   - تنظیم حداکثر پوزیشن 3
   - ریست آمار
   - حذف آمار

2. مشکل رد شدن سیگنال‌ها:
   - تشخیص جهت روزانه از حالت خیلی خشک خارج شد.
   - ورود 15M/5M منطقی‌تر شد.
   - امتیاز نهایی بعد از پاس کردن قفل‌های اصلی فقط برای نمایش است و سیگنال را دوباره بی‌دلیل رد نمی‌کند.
   - اگر سطح کامل روزانه پیدا نشود، TP پروجکشن حداقل 3٪ استفاده می‌شود.

قوانین اصلی باقی مانده‌اند:
- جهت اصلی 1D
- BTC و ETH باید هم‌جهت باشند
- TP فقط یکی
- SL فقط یکی
- حداقل TP برابر 3٪
- RR برابر 1:2
- Toobit Client دست‌نخورده
